# UniConnect Recife

UniConnect Recife é uma aplicação web desenvolvida em Django para criação, divulgação e gerenciamento de eventos universitários na cidade do Recife.

---

## Funcionalidades

- Criação de eventos universitários
- Visualização de próximos eventos
- Confirmação de presença
- Sistema de comentários
- Painel administrativo do Django
- Interface moderna e responsiva

---

## Tecnologias Utilizadas

- Python
- Django
- SQLite
- HTML5
- CSS3
- Git/GitHub

---

## Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Ylgner-Bolivart/uniconnect-recife.git
```

### 2. Entre na pasta do projeto

```bash
cd uniconnect-recife
```

### 3. Crie o ambiente virtual

```bash
python -m venv venv
```

### 4. Ative o ambiente virtual

```bash
venv\Scripts\activate
```

### 5. Instale o Django

```bash
pip install django
```

### 6. Execute as migrações

```bash
python manage.py migrate
```

### 7. Rode o servidor

```bash
python manage.py runserver
```
